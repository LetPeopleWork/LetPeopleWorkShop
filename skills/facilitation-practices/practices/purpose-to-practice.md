# Purpose-To-Practice (P2P)

- **Purpose:** Design the five essential, self-reinforcing elements of a resilient initiative together — Purpose, Principles, Participants, Structure, Practices — so the result is owned by everyone who will run it, not handed down.
- **When to use:** Launching a new initiative, team, project, or community and wanting it robust and collectively owned from day one. When a fuzzy "let's do this thing" needs to become a shared, durable design.
- **Group config:** Each element runs as its own 1-2-4-All (self -> pairs -> quads -> whole group), worked in sequence over one long session or several.
- **Timing:** ~120 min for a solid full pass (roughly 20-25 min per element via 1-2-4-All); often split across multiple sessions. Trim to ~75 min by tightening each element's rounds, but never skip an element.
- **Medium fit:**
  - In-person: five labeled wall areas, one per element; build each with stickies as you go, leaving them visible so later elements can react to earlier ones.
  - Miro: a frame per element, each with 1-2-4-All sub-frames; the five frames stay on one board so the whole design is visible at once.
  - Video: breakouts for the pair/quad rounds of each element, plenary harvest, one shared doc with five sections.
- **Steps:**
  1. **Purpose:** Why does this work matter? Ask "why" repeatedly to reach the deepest shared purpose. Do this via 1-2-4-All.
  2. **Principles:** What rules/constraints must we hold to serve that purpose? Must-dos and must-not-dos.
  3. **Participants:** Who must be involved for this to succeed — inside and outside the obvious circle?
  4. **Structure:** How will we organize, distribute power, and connect? What's the shape of the group?
  5. **Practices:** What concrete activities will we actually do, day to day, to move forward?
  6. Each element is built on the one before; revisit earlier elements if a later one exposes a gap.
- **Facilitator notes:** Start with Purpose and DO NOT rush it — every other element is derived from it, and a shallow purpose ("deliver the project on time") produces a brittle design. Drive the "why" deeper than the group wants to go. The five elements must end up mutually consistent; if Structure contradicts Principles, that's signal, not noise — loop back. It's completely normal to run this across multiple sittings; don't compress it into one rushed hour.
